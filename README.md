📊 Logic Overview
This project implements the FizzBuzz algorithm, a staple in programming education that demonstrates efficient use of conditional statements:

- **Range-Based Iteration:** Instead of starting at 1, this version allows users to define a custom starting and ending range.
- **Multi-Condition Evaluation:** The program evaluates numbers for divisibility by 3, 5, or both (15).
- **Output Priority:** Demonstrates correct logical nesting where the most restrictive condition (divisible by both 3 and 5) is checked first to ensure accuracy.

🛠️ Features
- **Dynamic Input:** Captures two user-defined integers to set the loop boundaries.
- **Clean Logical Flow:** Uses `if-else if-else` structures to ensure only one output per iteration.
- **Modulo Arithmetic:** Showcases the practical application of the `%` operator in data filtering.

🚀 Execution Guide
1. **Compile:** `g++ fizzbuzz_logic.cpp -o FizzBuzz`
2. **Run:** `./FizzBuzz`
3. **Test:** Enter `1` for the first number and `20` for the second to see the standard FizzBuzz output.
